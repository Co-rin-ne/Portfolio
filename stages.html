📇 Mes Stages

Les expériences professionnelles réalisées au cours de ma formation

🌸 Stage de première année — Service de sécurité de l'État

🌍 Présentation de l'organisme d'accueil

Dans le cadre de ma première année de BTS SIO SLAM, j'ai effectué un stage de six semaines au sein d'un service de sécurité de l'État gabonais, principalement à Libreville, la capitale du Gabon, avec une mission complémentaire à Franceville dans la province du Haut-Ogooué.

Cet environnement technique exigeant, ancré dans les enjeux stratégiques du pays, m'a permis de développer des compétences concrètes en administration réseau, virtualisation et cybersécurité, sous la tutelle de Mme AZIZET Emma Léa.

🖥️ Virtualisation — VMware ESXi

📍 Libreville, Gabon

Installation et configuration de VMware ESXi sur un serveur dédié. Création de l'architecture réseau virtuelle : 3 vSwitch (Management, Vidéo, Forensic), configuration des interfaces VMkernel et déploiement de 3 machines virtuelles sur des VLANs segmentés.

🔒 Pare-feu — OPNsense

📍 Libreville, Gabon

Découverte et configuration d'OPNsense, pare-feu open source basé sur FreeBSD. Installation sur machine virtuelle, configuration des interfaces réseau, VLANs, règles de filtrage, gestion du NAT et segmentation des flux réseau.

💻 Mise à jour d'une application web

📍 Libreville, Gabon

Correction d'une application de pointage (arrivées/départs/absences) codée en Python et JavaScript. Identification d'un bug sur la gestion des heures de départ, modification du code, tests en local et déploiement sur le réseau privé de l'organisme.

📹 Service Vidéo & Surveillance

📍 Libreville, Gabon

Immersion dans le service vidéo chargé de la surveillance nationale. Utilisation de Zabbix pour le suivi des caméras, intervention sur site pour remplacement de matériel, et reconnaissance de plaques d'immatriculation via l'application LPR.

📡 Service ROEM — Renseignement électromagnétique

📍 Libreville, Gabon

Découverte du Renseignement d'Origine Électromagnétique : interception de communications GSM, transcription et transmission au chef de secteur selon une chaîne de traitement coordonnée.

🛰️ Mission à Franceville — Interception satellitaire

📍 Franceville, province du Haut-Ogooué

Déplacement pour découvrir l'interception de communications via téléphones Thurayas (bandes L et C). Utilisation de GeoMap pour la géolocalisation et le suivi des terminaux satellitaires en temps réel.

🛠️ Technologies utilisées — SN1

VMware ESXi OPNsense Python JavaScript VS Code Zabbix LPR GeoMap GSM / ROEM Réseau / VLAN

🌺 Stage de deuxième année — Institut National de l'Audiovisuel

🎬 Présentation de l'organisme d'accueil

Dans le cadre de ma deuxième année de BTS SIO SLAM, j'ai effectué un stage de six semaines à l'INA (Institut National de l'Audiovisuel), établissement public à caractère industriel et social fondé en 1975, dont le siège est situé à Bry-sur-Marne (Île-de-France).

L'INA est l'une des premières ressources mondiales en matière de contenus audiovisuels numérisés. Il est chargé de la conservation, de la valorisation et de la transmission du patrimoine audiovisuel national. J'y ai intégré l'équipe de développement informatique, débutant le 12 janvier 2026.

🔐 Authentification LDAP

📍 INA, Bry-sur-Marne

Développement d'un module d'authentification via LDAP : validation des mots de passe, changement sécurisé avec politique de complexité (longueur minimale, caractères spéciaux, chiffres), et interface d'administration locale modernisée.

🚀 Application full-stack Angular / Node.js

📍 INA, Bry-sur-Marne

Conception et développement d'une application full-stack : front-end Angular (avec Material UI) et back-end Node.js. Développement des endpoints d'API REST et intégration de l'authentification LDAP dans l'interface utilisateur.

🐳 Déploiement Docker & Nginx

📍 INA, Bry-sur-Marne

Containerisation de l'application avec Docker et mise en place d'un reverse proxy Nginx. Configuration du middleware CORS pour les requêtes cross-origin et déploiement dans l'environnement de l'INA.

🧪 Tests & Débogage API

📍 INA, Bry-sur-Marne

Tests des endpoints API avec Postman, identification et correction des bugs. Mise en place de scénarios de test couvrant les cas nominaux et les cas d'erreur pour garantir la robustesse de l'application.

🛠️ Technologies utilisées — SN2

Angular Node.js JavaScript PHP LDAP Docker Nginx REST API GitLab Postman Material UI
