Mon CV numérique

Projet de découverte HTML

📋 Contexte du projet

Dans le cadre de ma formation BTS SIO SLAM, j'ai réalisé un CV numérique en HTML et CSS lors de mon apprentissage du développement web front-end. L'objectif était de créer une page web professionnelle et responsive présentant mon parcours, mes compétences et mes expériences de manière structurée et visuellement attractive.

Ce projet m'a permis de découvrir les fondamentaux du développement web : structuration HTML sémantique, mise en forme avec CSS, et adaptation responsive pour différents supports (desktop, tablette, mobile). C'était ma première réalisation concrète en tant que développeuse web.

🎯 Mission et réalisations

  • Découverte de l'utilisation de HTML
  • Découverte de l'utilisation de CSS
  • Mis en page avec image
  • Apprentissage de la structuration HTML/CSS

💡 Compétences développées

B1.1

Développer la présence en ligne de l'organisation

J'ai créé une page web professionnelle servant de vitrine numérique. Ce CV en ligne permet une présence web accessible 24h/24 et facilite le partage de mon profil auprès de recruteurs et d'entreprises.

B1.2

Gérer le patrimoine informatique - Structures HTML

J'ai structuré le document HTML de manière sémantique en utilisant les balises appropriées (header, section, article, footer) pour organiser l'information de façon logique et accessible. Cette structuration facilite la maintenance et l'évolution du code.

B1.3

Répondre aux incidents et aux demandes d'assistance - Design responsive

J'ai développé une interface responsive qui s'adapte automatiquement à tous les supports (ordinateur, tablette, smartphone) en utilisant les media queries CSS. Cela garantit une expérience utilisateur optimale quel que soit l'appareil utilisé.

⚠️ Difficultés rencontrées et solutions apportées

Difficulté 1

Problème : J'ai rencontré des difficultés pour positionner correctement ma photo de profil sur la page. L'image ne s'alignait pas comme souhaité avec le texte, créant un rendu peu professionnel avec des décalages et des espaces indésirables.

Solution : Après plusieurs recherches et tests, j'ai découvert les propriétés CSS Flexbox qui permettent un alignement flexible des éléments. J'ai utilisé display: flex sur le conteneur parent, puis align-items et justify-content pour positionner l'image et le texte côte à côte de manière harmonieuse. J'ai également ajouté des marges et du padding pour gérer précisément les espacements.

✅ Résultats et bilan

À l'issue de ce projet, j'ai réussi à créer un CV numérique entièrement fonctionnel et responsive. La page s'affiche correctement sur tous les supports (ordinateur, tablette, smartphone) et présente mes informations de manière claire et professionnelle. L'image de profil est bien intégrée et s'harmonise parfaitement avec le reste de la mise en page.

Ce projet m'a permis de maîtriser les fondamentaux du développement web front-end : structuration HTML sémantique, mise en forme CSS, gestion du responsive design avec les media queries, et manipulation du positionnement avec Flexbox. J'ai également développé ma capacité à rechercher des solutions de manière autonome face aux problèmes techniques rencontrés.

Ce que j'ai appris : Au-delà des compétences techniques, ce projet m'a fait comprendre l'importance de tester régulièrement son travail sur différents supports, d'optimiser les ressources (images) pour améliorer les performances, et de structurer son code de manière propre pour faciliter la maintenance future. Avec du recul, j'aurais planifié dès le départ la version mobile pour éviter de devoir restructurer mon code par la suite.

Évolutions possibles : Si je devais améliorer ce projet, j'ajouterais des animations CSS subtiles pour rendre la page plus interactive, j'intégrerais un formulaire de contact fonctionnel, et j'optimiserais encore davantage le code en utilisant des variables CSS pour faciliter les modifications de couleurs et d'espacements.
