03

Brasserie Serveur

Site web avec CRUD, base de données et calculateur de colorimétrie de la bière

📋 Contexte du projet

Dans le cadre de ma formation BTS SIO SLAM, j'ai développé un site web complet pour une brasserie artisanale. Ce projet avait pour objectif de créer une application web fonctionnelle intégrant une gestion de base de données, un système CRUD (Create, Read, Update, Delete) et une fonctionnalité originale : un calculateur de la colorimétrie de la bière.

Ce projet m'a permis de mettre en pratique mes compétences en développement web et de me familiariser avec la gestion de bases de données relationnelles dans un contexte professionnel concret.

🎯 Mission et réalisations

  • Développement du site web de la brasserie (front-end et back-end)
  • Création et gestion de la base de données des produits
  • Implémentation du système CRUD pour la gestion du catalogue bière
  • Conception d'un calculateur de colorimétrie basé sur l'échelle EBC
  • Interface d'administration pour la gestion des références

💡 Compétences développées

B1.1

Développer la présence en ligne de l'organisation

J'ai développé un site web professionnel servant de vitrine digitale pour la brasserie, permettant aux clients de découvrir les produits et aux gérants de les administrer facilement.

B1.4

Travailler en mode projet

Ce projet a nécessité une planification rigoureuse : conception de la base de données, développement front-end puis back-end, et tests fonctionnels à chaque étape.

B2.1

Concevoir et développer une solution applicative

J'ai conçu l'architecture de l'application, modélisé la base de données et développé l'ensemble des fonctionnalités CRUD ainsi que le module de calcul de colorimétrie.

🛠️ Technologies utilisées

Front-end

HTML CSS JavaScript

Back-end & Base de données

SQL PHP CRUD

⚠️ Difficultés rencontrées

Calcul de la colorimétrie de la bière

Problème : Implémenter un calculateur de colorimétrie basé sur l'échelle EBC nécessitait de comprendre les formules brassicoles et de les traduire en algorithme.

Solution : J'ai effectué des recherches approfondies sur les méthodes de calcul brassicoles, puis implémenté la formule en JavaScript avec une visualisation couleur du résultat estimé.

Gestion des relations entre tables SQL

Problème : La modélisation des relations entre les tables s'est avérée complexe pour garantir l'intégrité des données.

Solution : J'ai utilisé les clés étrangères et les contraintes SQL pour assurer la cohérence des données, et j'ai testé chaque opération CRUD.

✅ Résultats et bilan

Ce projet a été l'un des plus complets de ma première année. J'ai développé une application web fonctionnelle de bout en bout, depuis la conception de la base de données jusqu'à l'interface utilisateur finale.

Ce que j'ai appris : L'importance de bien modéliser ses données en amont, la rigueur nécessaire pour les opérations CRUD, et la satisfaction de livrer une application concrète et utilisable dans un contexte métier réel.
